# React Components Challenge

This project is the first challenge from the React course by [Tarmeez Academy](https://www.youtube.com/watch?v=80s3Rp64n5o&list=PLYyqC4bNbCIdSZ-JayMLl4WO2Cr995vyS&index=16).

## Getting Started

- Install dependencies:

```bash
npm install
```

- Run the development server:

```bash
npm run dev
```
